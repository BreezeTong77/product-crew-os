# Product Crew OS v0.2.0

发布日期：2026-07-12

## 这次是什么变化

运行时从 Ruby 全量迁到 Python + LangGraph。现在只有一条流程控制链：

```text
Input Scope Gate
-> Project Context
-> Retrieval Evidence
-> Stage / SOP / Skill Route
-> Skill Guard
-> Artifact
-> Review Interrupt
-> Review Summary
-> User Decision
-> Revision / Re-review
-> Gate
-> Project Assets Export
```

## 删除了什么

- Ruby Runtime CLI。
- Ruby Coze Bridge。
- Ruby 路由、RAG、OCR、Skill executor 和测试 runner。

## 新入口

- `runtime/pco_runtime.py`
- `runtime/pco_coze_bridge.py`
- `runtime/langgraph_runtime/adapters.py`
- `tests/validate-package.py`
- `tests/run-release-gate.py`

## 验收边界

发布门禁包含 44 条 SOP 路由控制与 L45-L50 Bad Case。它不会把模板、hash embedding、测试回调或未部署 MCP 写成真实执行。
