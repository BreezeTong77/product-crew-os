# Product Crew OS v0.2.1

发布日期：2026-07-12

## 本次更新

SOP 命中后，Skill 不再由调用方“报一个成功结果”。LangGraph 会自己执行 primary / fallback Skill，保存原始输出，并签发与项目、阶段和输出文件绑定的回执。

```text
Stage / SOP 命中
-> 图内执行 Skill
-> 原始输出落盘
-> 签名回执
-> Artifact
-> Review
-> 用户决策
-> Gate
```

## 已验证

- 49 份随包 Skill 都能被执行器发现。
- `research_plan -> product-discovery` 会真实执行本地脚本并把原始输出写入 Artifact。
- 本机 Ollama 环境下，`mvp_scope -> scope-cutting` 不可用时会真实执行备用 `shape-up`，再由图签发回执。
- 伪造的 `skill_execution` 会被 Coze Bridge 拒绝，不能绕过 Stage Gate。

## 仍需部署的能力

Figma、Pencil 等 MCP Skill 仍需要真实连接目标工具和用户写入授权；未部署时会阻塞，不会伪造外部文件。
