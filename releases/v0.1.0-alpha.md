# Product Crew OS v0.1.0-alpha

发布日期：2026-06-26

## 概要

`v0.1.0-alpha` 是 Product Crew OS 的第一个可迁移发布版本：它是一套面向产品经理工作流的 AI Product Coach 规则包。

本版本重点是让产品机制可以迁移、复用和继续迭代：

- Workflow-first 编排。
- Skill 路由。
- 细颗粒 Workflow SOP 库。
- 阶段门控式 stakeholder 评审。
- 子 Agent 真实调用契约和 invocation ledger。
- 子 Agent 长期记忆 runtime 契约和 `memory_snapshot`。
- Artifact Workspace。
- 记忆边界规则。
- 默认可见主控教练 profile：`甜心教练-董董`，魅力型领袖，思虑周全，亲和力拉满。
- 可配置的产品团队角色人格。
- 可配置的子 Agent 团队性格和团队风格 overlay。
- 用户自有 skill、月度 skill 发现和 MCP 可选适配规则。
- 能力地图、三种使用模式和自然语言触发语。
- 第三方 PM 能力包适配规则。
- Deep Artifact Pack、低保真原型、技术任务拆解和测试场景的最小可用能力。
- 原型增强路径：image 概念图 -> HTML Demo -> Pencil / Figma 可编辑原型。

## 包含内容

- `product-crew-os-skill/`
- `docs/product-rules.md` 中的产品规则。
- `docs/portability-manifest.md` 中的可迁移清单。
- 默认主控教练人格配置，并明确支持用户覆盖。
- `product-crew-os-skill/references/workflow-sop-library.md` 中的完整 PM 流程 SOP。
- `product-crew-os-skill/references/subagent-invocation-contract.md` 中的真实调用与模拟视角边界。
- `product-crew-os-skill/references/subagent-memory-runtime-contract.md` 中的角色记忆读取、压缩注入和反哺规则。
- Deep Artifact Pack、低保真原型、技术任务拆解和测试场景的可编辑模板。
- 低保真到较高保真原型的推荐生成路径。
- Skill 和工具生态规则，包括月度 skill 推荐和用户自有 skill overlay。
- 能力地图、使用方式和第三方能力包适配参考。
- MCP 作为用户既有软件工作流的可选适配层说明。
- GitHub 发布检查清单。
- README、LICENSE、CHANGELOG。

## 不包含内容

- 真实项目 workspace。
- 用户偏好记忆。
- 项目记忆。
- 要求用户必须保留默认主控教练名称或性格。
- 私有 PRD、访谈、会议纪要、客户数据或公司内部材料。
- 支付或 SaaS 实现。
- 针对所有 artifact 类型的全自动外部工具执行能力。

## 推荐安装方式

将 `product-crew-os-skill/` 复制到你的 Codex skills 目录。

```text
~/.codex/skills/product-crew-os/
```

然后调用：

```text
$product-crew-os
```

## 升级说明

这是第一个公开 alpha 版本。后续版本可能调整：

- 项目状态 schema。
- 团队风格 overlay schema。
- Skill overlay schema。
- 阶段分类。
- 回归测试预期。

如果你依赖当前文件结构，请固定使用该版本。

## 隐私说明

不要把本地项目 workspace 或真实用户材料提交到这个仓库。Product Rule Memory、User Preference Memory 和 Project Workspace Memory 必须分开保存。
